MTBS Burned Area Boundaries Shapefile
======================================
Download from: https://www.mtbs.gov/direct-download
File needed: "Burned Area Boundaries" Shapefile -> mtbs_perimeter_data.zip (~150 MB full CONUS)
Extract to this directory.

The ingester looks for: data/mtbs/mtbs_perimeter_data.shp

Coordinate system: NAD83 (EPSG:4269) -- MtbsIngester reprojects to WGS84 automatically.
